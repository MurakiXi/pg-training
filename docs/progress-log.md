- `currentQuestionIndex++` は、現在の数値を1つ増やす処理である。
- 固定値を代入する処理と、現在の値を更新する処理は意味が違う。
- 問題番号を更新した後、画面表示を変えるには `renderQuestion` をもう一度呼ぶ必要がある。
- `currentQuestionIndex` は、現在どの問題を表示しているかを表す状態変数として使える。
